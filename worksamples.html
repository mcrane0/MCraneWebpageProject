Journal Guide - Mental Health Journaling Program

In my sophomore year for my AP Computer Science Principles class, as part of the AP Exam, we were assigned the Create Task, where we create some sort of program using the different methods and strategies that we learned. For this, I decided to make a mental health journal using Python that guides you through how to express yourself, something that I find very important. This would be aptly named “Journal Guide”. Due being unable to create a program with a graphical user interface (GUI), I worked to make an intuitive command-line text editor, the heart of Journal Guide. To save the journals that users created, I made my own file format, .jgd, that journals could be saved to and opened from. So they can also be saved as an easy to read and universal format, they are given the option to save as a .txt (text) file. Below is a video of me using Journal Guide. Through writing all 312 lines of code (some can be seen below), I learned more about creating an experience that's welcoming for the user, coming up with creative solutions to the problems faced in coding, and more about the Python language and using it in general.


A Christmas Story: The Musical

Since middle school, I have always been a big fan of my friends' roles in theatre, and in my senior year, I finally joined them in the Barren County High School Drama Club. I didn't join to act, but instead to bring them my ability with technology. When I started helping, Ms. Gardner, the director, asked me to hook up a camera set up behind the audience to two TVs backstage, so the others backstage could see what’s happening onstage. I expected she would give me a modern camcorder that I'd use with a straightforward connection to the TVs, until she handed me a camcorder from 2001, seen in the first picture to the left. Still, I worked with her idea and found a converter for the camcorder’s analog video signal to a modern digital signal, used what I learned in Helpdesk to create two very long cables, and successfully gave everyone backstage a view of what was happening onstage. My work with cameras didn't stop there; I captured scenes acted live backstage with a different albeit much more modern camcorder, and displayed them to the screens in the auditorium. Below is a picture of me testing out connection to the screens. To give even more to the show, I designed cards which I would display on the screens between acts, before, and after the show, like the one for the start of Act Two seen below. I had so much fun through all the work I did on this production, and I'm looking forward to doing it again in the spring for Cinderella.


mcrane.website

The website you are currently looking at is a work sample. This was part of an assignment where I had to create a website about myself to then present to a panel. Chances are, you reading this right now are part of said panel.

Most people used Wix to create their website for this assignment. Given that the point of this website was to demonstrate how we're college and career ready and fit the "Profile of a Trojan", I decided to take a more creative approach and make my website the way I'm going to be pursuing a career in: programming.

I wrote my website from scratch using HTML and CSS. All styles and designs were made by me, with exception of some colors taken from color palette websites and the Visual Studio Code (a code editor) color theme to give a programming feel to the website. It was difficult to create all the designs I wanted from just code, especially compared to something you graphically interact with like Wix, however anything that went wrong on my website was something that I could eventually fix, and not be helpless against whatever issues Wix had with its buggy system.

To make the website accessible, I uploaded the code to GitHub (a place for uploading code), which has a feature allowing me to publish the website, and configure it with a low-cost domain, "mcrane.website", and now my website is up and on the web.

The full code for my website can be viewed on GitHub, and some of it for this work samples page can be seen in a picture to the right.
