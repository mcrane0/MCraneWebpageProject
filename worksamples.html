Journal Guide - Mental Health Journaling Program

In my sophomore year for my AP Computer Science Principles class, as part of the AP Exam, we were assigned the Create Task, where we create some sort of program using the different methods and strategies that we learned. For this, I decided to make a mental health journal using Python that guides you through how to express yourself, something that I find very important. This would be aptly named “Journal Guide”. Due being unable to create a program with a graphical user interface (GUI), I worked to make an intuitive command-line text editor, the heart of Journal Guide. To save the journals that users created, I made my own file format, .jgd, that journals could be saved to and opened from. So they can also be saved as an easy to read and universal format, they are given the option to save as a .txt (text) file. To the right is a video of me using Journal Guide. Through writing all 312 lines of code (some can be seen to the left), I learned more about creating an experience that's welcoming for the user, coming up with creative solutions to the problems faced in coding, and more about the Python language and using it in general.
